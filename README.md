# Disemvoweller

January 21, 2016

##### By Molly Curtin &amp; Abby Rolling

### Description

This website takes a user-submitted word or phrase, replaces the vowels with dashes, and returns the result.

##### Technologies Used

* HTML
* CSS
* JavaScript
* jQuery
* Mocha
* Chai
* BDD
* Git
